Backing up projects

A backup file is a Zip file that contains the data and text from a FieldWorks project.

  • Invest the time to back up projects frequently, copy the backup files to a compact disc or USB flash drive, and then store the files in a safe location.

  • To recover a lost project, restore it from a backup file.
