Visual demos

A demonstration shows you the procedure to perform a task. In the visual demos:

  • You read explanations of the procedure.

  • You follow along the steps where you click a command on a menu, a button in a dialog box, or text in a view.

    You will not need to double-click, right-click, press keys, or type text.

    The screen shots show only the relevant part of the Translation Editor window.

To display the index page: On the Help menu of the Translation Editor window, click Demos.

Each demo consists of slides. A Web browser displays each slide as a Web page. Your computer does not need to be connected to the Internet.

To go to the next slide of a demo, do one of the following:

  • Click where a text box points in the screen shot of Translation Editor.

  • Click Next or Index at the upper right of the slide.

After the last slide of a demo, the browser displays the index page. You can:

  • View another demo.

  • Close the browser tab or window.

You can click the Back button in the browser window:

  • If you need to go to the previous slide of a demo.

  • To return to the index page when you have finished reading this page.
