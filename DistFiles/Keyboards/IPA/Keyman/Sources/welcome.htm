Martin Hosken and Lorna A. Priest,

IPAUni12.kmx is a mnemonic keyboard definition which enters Unicode values for IPA. This version of the keyboard supports a mnemonic layout. This allows a user to type with any kind of keyboard, rather than just a US keyboard.

Keyboard Layout

The keyboard layout is described in terms of an IPA chart rather than a keyboard. This is because many base characters are typed as a sequence of a letter followed by one of <, > or = which are characters used to change a base character to another base character. Diacritics are typed as sequences of an appropriate key. Details of the keyboard layout may be found in the "IPA Unicode 5.1 (ver 1.2) KMN.pdf" document in the section on Keyboard Layout. (PDF, requires Adobe Reader)

Keyboard Installation

Keyman 6 or later is needed for using the keyboard. Keyman's latest version is available from here. If you will be writing or modifying keyboards, you will need the Developer version. Otherwise, get the regular version.

Keyman Install

    1. Uninstall any previous versions of Keyman
    2. Install Keyman, following the installation instructions provided with Keyman.
    3. If you haven't run Keyman for the first time yet, go to Start / Programs / Tavultesoft Keyman (or go to whatever location you chose during install) and click Keyman to start the program.
    4. This will briefly show an 'About' screen and place the Keyman icon in your system tray (bottom left of screen).
    5. Register your copy by right-clicking this icon, then click About / Register.
    6. Right click the Keyman icon, and click 'Start with Windows' if you want Keyman to be available each time you start up.

Keyboard Install

If you downloaded Unicode keyboard installation package IPAUni12.kmp:

    1. Install the keyboard by double-clicking the keyboard file: IPAUni12.kmp.
    2. Then click Install to install the keyboard file.
    3. Read documents related to this keyboard at: Start / All Programs / IPA Unicode 5.1 (ver 1.2) KMN

If you downloaded the Unicode keyboard zip file IPAUni12.zip:

    1. Unzip the files to an appropriate place
    2. Install the keyboard by double-clicking the keyboard file: IPAUni12.kmx.
    3. Then click Install to install the keyboard file.

Using the keyboard

Using Keyman Selector

    1. Click on the Keyman keyboard selector in the taskbar (bottom right of screen).
    2. Select the keyboard you want from the menu.

Using Keyman keyboards in place of Windows keyboards

    1. Start Keyman Configuration.
    2. Select the Languages tab.
    3. In the table that appears, select the language for which you wish to always use a Keyman keyboard.
    4. Click in the last column or press F2 to select the Keyman keyboard.
    5. Click the OK button to save your selection.
    6. Click the Windows language selector, and select the language you just associated with your keyboard.
