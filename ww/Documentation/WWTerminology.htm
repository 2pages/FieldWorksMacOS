Brief Glossary of WordWorks Terminology

Term Definition
DataTree Tree control used in xCore to display and allow editing of data(?). Each data item is in a slice.
FDO FieldWorks Data Objects
FLexicon FieldWorks Lexicon
FXT FieldWorks XML templates (FXT) are XML files which FieldWorks can use to export data out of its database into an XML file
GAFAWS Grime's Algorithm for Analyzing Words - a tool to help a user figure out an inflectional template based on a set of hand-parsed wordforms.
IText Interlinear Text tool
JIRA Bug/enhancement tracking software package
jtview "John Thomson" views code editor used in XDEs
LexEd Lexicon editor
M3 / M3 Morphology our third generation conceptual model of morphology
ME Morphology Editor
MGA Morphosyntactic Glossing Assistant - a tool that helps users choose accurate, standard and consistent glosses. Behind the scenes, it also assigns feature structures for the glosses (at least for those we know about).
ParseBench Tool to faciliate parsing sets of words
PC-PATR SIL's syntactic level parser tool, based on the PATR II formalism
slice a row in a DataTree
XAMPLE "eXperimental AMPLE" - the AMPLE morphological parser augmented with a PC-PATR word grammar. We're using XAMPLE as our Stage 1 morphological parser.
xCore xCore is a DotNet-based framework. The goal of this framework is to allow us to create applications by writing an XML file which collects various components and wires them together. So with XCore, you can set up the menu items, toolbars, and sidebars. The actual content-area of the application, where you see real data and do real work, is beyond the scope of XCore, except that it does provide some useful pieces for allowing different components to talk to each other and (eventually) for applications to talk to each other.
XDE XML-based Data Entry description(?)
Xerox Parser A morphological parser developed by Xerox Parc that allows one to write standard generative phonological rules. We hope to use this as our Stage 2 and 3 morphological parser.
xWorks FieldWorks-specific xCore application(?)
