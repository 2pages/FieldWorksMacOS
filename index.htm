In order to access the "Classes and methods" link from on-center machines, set up Internet Explorer as follows:

  • Tools...Internet Options
  • Connections tab
  • Click LAN Settings
  • "Use a proxy server" should be checked as well as "Bypass proxy server for local addresses"
  • Click Advanced
  • Set HTTP: to msproxy.sil.org, port 80, and check "Use the same proxy server for all protocols"
  • In the Exceptions pane, add *.sil.org to the "Do not use proxy server for addresses beginning with" list.

Also, on-center users should make the following settings (Instructions apply to Windows 2000):

  • Start...Settings...Network and dial-up connections
  • Right-click "Local Area Connection" and choose properties
  • Double-click "Internet Protocol (TCP/IP)"
  • "Obtain an IP address automatically" should be checked
  • "Use the following DNS server addresses" should be checked, and the Preferred DNS server should be 172.20.1.4 and the Alternate DNS server should be 172.20.1.3.
